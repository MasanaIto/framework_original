# framework_original
オリジナルのphpフレームワーク

開発する際は以下の項目を作る必要がある。

Applicaitonクラス
  -アクションに合わせたルーティング定義
  -接続するデータベース指定（DbManagerクラスを操作）
  -ログインアクションの指定
  
index.php
  -Applicationの呼び出しと実行
  
Controllerクラス（コントローラ）
  -役割に合わせて小クラスの作成
  -作成する外面に合わせたアクション定義、処理の実装
  -ログインが必要なアクションの指定
  
DbRepositoryクラス（モデル）
  -データベース上のテーブルごとに小クラスの作成
  -データベースアクセス処理の実装
 
 ビューファイル（ビュー）
  -アクションに合わせたHTMLの記述
  -レイアウトファイルの記述
